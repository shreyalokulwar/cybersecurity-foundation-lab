# Lab Setup Report

## Objective

To create a cybersecurity lab environment for learning networking, Linux, cryptography, and penetration testing concepts.

## Environment Details

### Virtualization Software
VirtualBox

### Attacker Machine
Kali Linux

### Target Machine
Metasploitable2 / DVWA

## Tools Installed

- Wireshark
- Nmap
- Burp Suite
- Netcat

## Network Configuration

A Host-Only Adapter was configured to create an isolated testing environment between Kali Linux and the target machine.

Benefits:
- Safe Testing
- No Internet Exposure
- Controlled Environment

## Verification

The following checks were performed:

- Kali Linux booted successfully.
- Target machine was reachable.
- Ping communication worked.
- Wireshark packet capture tested successfully.
