# Networking Basics

## TCP/IP Model

1. Application
2. Transport
3. Internet
4. Network Access

## Important Protocols

### TCP
Reliable communication protocol.

### UDP
Fast, connectionless communication protocol.

### HTTP
Used for web communication.

### HTTPS
Secure version of HTTP using SSL/TLS.

### DNS
Converts domain names into IP addresses.

## IP Addressing

Example:

```text
192.168.1.10
```

## NAT

Network Address Translation enables multiple devices to share one public IP address.

## Subnetting

Subnetting divides large networks into smaller networks.

## Conclusion

Networking knowledge is critical for understanding data communication and network security.
