# Tool Familiarization

## Wireshark

Wireshark is a packet capture and network analysis tool.

Uses:
- Traffic Monitoring
- Packet Inspection
- Network Troubleshooting

## Nmap

Nmap is a network scanning tool.

Example:

```bash
nmap 192.168.1.1
```

Uses:
- Host Discovery
- Port Scanning
- Service Detection

## Burp Suite

Burp Suite is used for web application security testing.

Features:
- Proxy
- Repeater
- Intruder
- Scanner

## Netcat

Netcat is a networking utility for reading and writing network connections.

Example:

```bash
nc -lvnp 4444
```

Uses:
- Port Listening
- Data Transfer
- Network Debugging

## Conclusion

These tools are essential for ethical hacking and security assessments.
